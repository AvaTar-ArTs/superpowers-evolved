# Tier 1 — The Iron Foundation

**Install these first. Always.** Every other skill assumes you have these internalized.

## systematic-debugging v3

Four-phase root cause analysis. The Iron Law: *no fixes without root cause investigation first.*

What v3 adds over vanilla:
- Multi-component evidence gathering (CI → build → signing pipelines)
- Architecture-questioning rule: 3+ failed fixes = question the architecture, not keep patching
- 12-item Red Flag table to short-circuit rationalization
- User signal detection ("Is that not happening?" = you assumed without verifying)

## verification-before-completion v3

Evidence before claims, always. The Gate Function.

What v3 adds over vanilla:
- 5-step Gate Function (IDENTIFY → RUN → READ → VERIFY → CLAIM)
- Common Failures table: what each claim type actually requires vs what's insufficient
- Rationalization Prevention: every excuse mapped to reality
- Agent delegation verification: never trust an agent's "success" report
