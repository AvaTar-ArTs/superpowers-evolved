# Tier 2 — Workflow: Idea → Shipped Feature

**Unlocks after Tier 1.** Bring ideas from concept to completed, reviewed code.

| Skill | Role in the chain |
|-------|-------------------|
| `brainstorming` v3 | Idea exploration → design doc |
| `writing-plans` | Design doc → implementation plan |
| `executing-plans` | Plan → working code |
| `finishing-a-development-branch` | Working code → merged PR |
| `test-driven-development` | Red-green-refactor cycle |
| `using-git-worktrees` | Parallel feature branches |

The chain: **brainstorm → plan → execute → finish**. Each skill feeds the next.
Tier 1 skills (`systematic-debugging`, `verification-before-completion`) are called
at every stage.

## What v3 adds over vanilla

- `brainstorming` v3: Hybrid mode balancing strict vs flexible exploration
- All skills include Qwen/Gemini/Cline-specific tool invocation patterns
